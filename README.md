📌 Visão Geral
SpaceGuard AI é uma prova de conceito (POC) que demonstra como a Inteligência Artificial pode transformar dados espaciais em ações concretas na Terra. O sistema integra:

Visão Computacional (YOLO) para detecção de anomalias em imagens de satélite
Processamento Serverless (AWS Lambda) para escalabilidade automática
Armazenamento em Nuvem (S3 + DynamoDB) para dados estruturados e não-estruturados
Frontend Interativo (React) para visualização e tomada de decisão
APIs Públicas (NASA/INPE) para ingestão de dados em tempo real
🎯 Problema Resolvido
Como a IA pode transformar a nova economia espacial e gerar impacto positivo na Terra?

Resposta: Através do monitoramento autônomo de áreas críticas (florestas, áreas agrícolas, zonas costeiras) usando imagens de satélite, detectando mudanças e gerando alertas acionáveis em tempo real.
